# 员工休假制度

## 1. 总则
为规范员工休假管理，保障员工合法权益，根据国家相关法律法规，制定本制度。

## 2. 休假类型

### 2.1 年假
员工连续工作满1年的，享受带薪年假。工作年限1-10年的，年假5天；10-20年的，年假10天；20年以上的，年假15天。

### 2.2 病假
员工因病需要治疗和休养的，可申请病假。病假期间工资按照国家规定执行。

### 2.3 事假
员工因私事需要请假的，可申请事假。事假期间不支付工资。

## 3. 申请流程

### 3.1 年假申请
员工需提前3个工作日向直属上级提交书面申请，经审批后方可休假。

### 3.2 病假申请
员工需在病假开始当天通知直属上级，并在3个工作日内提交医院证明。

## 4. 其他规定
年假最小使用单位为半天，当年未休完的年假可结转至次年3月底。
