# IT设备申领流程

## 1. 适用范围
本流程适用于公司全体员工申领办公电脑、显示器、键盘鼠标、手机等IT设备。

## 2. 设备标准

| 岗位类型 | 电脑标准 | 显示器 | 预算上限 |
|---------|---------|--------|---------|
| 普通员工 | ThinkPad T系列 / MacBook Air | 24寸 1080P | 8000元 |
| 开发工程师 | ThinkPad P系列 / MacBook Pro 14寸 | 27寸 4K | 15000元 |
| 设计师 | MacBook Pro 16寸 | 27寸 5K | 20000元 |
| 管理层 | ThinkPad X1 / MacBook Pro 16寸 | 27寸 4K | 18000元 |

## 3. 申领流程

### 3.1 新员工入职
1. HR在入职前3个工作日提交IT设备申请
2. IT部门根据岗位标准准备设备
3. 入职当天由IT进行设备发放和初始化配置
4. 员工签署《IT设备领用确认书》

### 3.2 在职员工更换
1. 员工在ITSM系统提交设备更换申请
2. 直属上级审批
3. IT部门评估设备状况：
   - 使用满3年：直接更换
   - 使用未满3年：需说明更换原因
4. IT部门经理审批（预算超过5000元）
5. 设备到位后，员工在3个工作日内完成数据迁移并归还旧设备

### 3.3 临时借用
1. 填写《设备临时借用申请表》
2. 部门负责人签字
3. 到IT仓库领取
4. 借用期限不超过15个工作日

## 4. 设备归还与报废
- 离职员工须在最后工作日归还所有IT设备
- 报废设备由IT部门统一处理，员工不得私自处置
- 设备丢失需按折旧价值赔偿

## 5. 常见问题

**Q: 新买的电脑不满意可以更换吗？**
A: 设备发放后7天内可申请更换，超过7天按在职更换流程处理。

**Q: 可以自己买设备报销吗？**
A: 不可以。所有IT设备必须通过公司统一采购渠道。
